How to Play

  • Use commands to control the game. Commands are only accepted at the beginning of messages.
  • Anarchy and Democracy are modes that affect how the game accepts commands. The game is always in one mode.
  • In Democracy, each viewer gets one vote and one downvote. Downvote by putting "no" or "-" in front of your command.
    Credits to the downvote idea go to Bga_mikengo.

Basic Commands

  1. START - start
  2. SELECT - select, sel
  3. A
  4. B
  5. UP - up, north
  6. DOWN - down, south
  7. LEFT - left, west
  8. RIGHT - right, east
  9. HOLD - hold, stay, stop
  10. ANARCHY - start
  11. DEMOCRACY - start

Advanced Commands

  1. @set goal, followed by a message, allows any user to set the goal (visible in the stream). The goal cannot be set more than once every 10 minutes.
  2. @clear goal, when said by the majority of active users, will replace the current goal with "[EXPLETIVE DELETED]". After the goal is cleared, the next goal cannot be set for 5 minutes.
  3. @get goal, commands Twippy to repeat the goal in chat. This is useful when the goal is set with a hyperlink. This command can only be called once a minute.
